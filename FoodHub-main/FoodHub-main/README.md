# 🍔 FoodHub - Online Food Ordering System

## 📌 Project Overview

FoodHub is a Java Full Stack web application that allows users to browse restaurants, view menus, add food items to a cart, and place orders online. The application provides a simple and user-friendly interface similar to food delivery platforms like Zomato and Swiggy.

---

## 🚀 Features

### 👤 User Module
- User Registration
- User Login
- View Profile
- Browse Restaurants
- Search Restaurants
- View Restaurant Menus
- Add Food Items to Cart

- Remove Cart Items
- Checkout
- Place Order

### 🍽 Restaurant Module

### 🛒 Cart Module

- View Cart

## ⚙ How to Run the Project

1. Clone or download the project.
2. Open Eclipse IDE.
3. Import the project.
4. Configure Apache Tomcat 10.
5. Create the MySQL database.
6. Update database credentials in the JDBC connection file.
7. Run the project on Tomcat.
8. Open:

```
http://localhost:8080/JEE1/
```
